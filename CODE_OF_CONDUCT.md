# Code of Conduct

StockTracker 希望保持一个友好、尊重、务实的开源协作环境。

## 我们鼓励

- 友善、清晰、具体地沟通。
- 对问题本身提出反馈，而不是攻击个人。
- 尊重不同经验水平的贡献者。
- 在讨论投资、数据和 AI 输出时保持审慎。
- 主动保护自己和他人的隐私数据。

## 我们不接受

- 人身攻击、骚扰、歧视或威胁。
- 泄露他人的 API Key、交易记录、数据库或个人资产信息。
- 在公开讨论中发布敏感安全漏洞细节。
- 故意提交恶意代码、后门或破坏性变更。

## 处理方式

维护者可以对违反行为进行提醒、编辑、隐藏、关闭 Issue / PR，或在必要时限制参与。

如果你发现安全或隐私问题，请优先参考 [SECURITY.md](./SECURITY.md)。
